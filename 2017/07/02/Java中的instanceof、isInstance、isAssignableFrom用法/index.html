说明：

instanceof的使用方法为obj instanceof class 如果 class obj1 = obj成立的话返回true，否则返回false。这里有两种特殊情况需要说明一下：

  1. 对于A ab = new B()这种情况，因为ab.getClass()的类型为B所以ab instanceof A和ab instanceof B都为true

  2. 当obj为null时，总是返回false。

    isInstance的使用方法为class.isInstance(obj),如果class obj1 = obj成立的话返回true，否则返回false。上述两种特殊情况仍然适用于isInstance。

isAssignableFrom的使用方法为classA.isAssignableFrom(classB), 如果B是A的子类，则返回true，否则返回false。 注意，这里classA和classB都不能为null，否则会抛出异常。
